# USMA Courses

This repository contains Google Colab notebooks for various USMA courses. Each notebook is organized by course.

## Getting Started

To use these notebooks, open the respective `.ipynb` files in Google Colab or Jupyter Notebook.

## Requirements

- Python 3.x
- Relevant libraries as specified in each notebook.
